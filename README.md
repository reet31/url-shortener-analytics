# clipTrace
A full-stack URL shortener with real-time click analytics. Shorten links, track every click, and understand your audience.

## ✨ Features

- URL Shortening : paste any long URL and get a clean short link instantly
- Custom Aliases : choose your own slug instead of a random code
- Click Analytics : every click is tracked with browser, device, country and referrer data
- Per-link Dashboard : see top performers, average clicks, and trends
- Expandable Rows : click any link to reveal its full analytics breakdown
- IP Geolocation : detects the country of every visitor automatically
- UA Parsing : identifies browser and device type from user agent


## 🛠 Tech Stack
 ### Frontend
React,Tailwind CSS,React Router

### Backend
Node.js,Express

### Database
MongoDB Compass,Mongoose

### Packages
- nanoid — unique short code generation
- ua-parser-js — browser and device detection
- ip-api.com — IP geolocation (free tier)
